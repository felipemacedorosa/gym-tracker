# Gym Tracker

A simple local gym workout tracker. No installation required.

## How to run

1. Open the `gym-tracker` folder.
2. Double-click `index.html` — it opens in your browser.
3. That's it.

## How to use

- Click **+ Add Workout Day** to create a day (e.g. Push Day, Leg Day).
- Inside a day card, click **+ Add Exercise** to log an exercise.
- Fill in name, weight, sets, reps, and optional notes.
- Use the ✏️ button to edit, 🗑️ to delete.
- All data is saved automatically in your browser's localStorage.

## Limitations

- Data is stored in your browser only — clearing browser data will erase it.
- No export or backup feature.
- No graphs or history comparison.
